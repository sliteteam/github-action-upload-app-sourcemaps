# Upload GraphQL schema

Upload sourcemaps to Bugsnag

## Usage

```yaml
inputs:
  imageTag:
    description: "The app docker image tag"
    required: true
  assetsPublicPath:
    description: "The assets public path once deployed"
    required: true
  bugsnagApiKey:
    description: "The BugSnag API key"
    required: true
```
